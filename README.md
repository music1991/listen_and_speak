# Listen & Speak

**Listen & Speak** is an interactive web application designed to help users learn and practice pronunciation through audio.  
The platform allows users to listen to the pronunciation of alphabet letters, example words, and custom text in real time.

It supports pronunciation in **Spanish and English**, making it suitable for language learners of different levels.

---

## Features

- Alphabet pronunciation with audio feedback  
- Example words for each letter  
- Support for Spanish and English pronunciation  
- Custom text input with real-time pronunciation  
- Simple and intuitive user interface  
- Educational and beginner-friendly approach  

---

## Use Case

This project is intended for:

- Language learners practicing pronunciation  
- Educational demonstrations of text-to-speech features  
- Exploring interactive learning experiences using web technologies  

---

## Tech Stack

- React  
- TypeScript  
- JavaScript Web Speech API (SpeechSynthesis)  
- Modern CSS / UI styling
